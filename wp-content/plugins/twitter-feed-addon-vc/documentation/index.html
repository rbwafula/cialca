A) Installing Instructions - top

  1. First and foremost, download and install WPBakery Page Builder
  2. Get the twitter-feed-addon-vc.zip file and unzip it.
  3. Upload all the files to the /wp-content/plugins/twitter-feed-addon-vc
  4. Activate the plugin through the 'Plugins' menu in WordPress.
  5. For customizing the plugin's settings, click on Twitter Feed Addon menu in Wordpress left admin menu.
  6. To display the twitter feed in the frontend, please use the backend or frontend editor of WPBakery plugin.

Twitter Feeds Settings

For convenience and since Twitter allows this, we have made this feature so that the Admin can set a Twitter App once and other users can show their twitter feed by just using their Username. If you are a not the admin and your twitter feed do not work, please contact your site admin to confirm that they have set the Twitter App.

Steps to configure Twitter:

  • First you need to go to https://apps.twitter.com/
  • You need to login to Twitter if you are not already logged in.
  • Click on Create New App
  • You will need to fill a Create an Application form where you need to specify the application Name, Description, Website, a callback URL(optional) and agree to the Developer Agreement. Once you have filled all the form fields click on create your Twitter Application.
  • Congratulation you have created a Twitter App! To get your Consumer Key(API Key) and Consumer Secret(API Secret), go to the Keys and Access Tokens Tab.
  • Check the Application setting to get your Consumer Key(API Key) and Consumer Secret(API Secret).
  • Look for Create my Access Token button to generate your Access token and Access token secret. Click on the button to view your Access token and Access token secret.
  • And you are done! Now you can use API Key, API Secret, Access token and Access token secret.

Feeds

Once you have filled all the necessary settings you can now fetch Twitter feeds.

Twitter Feeds

The WPBakery Page Builder plugin allows you to add elements into the page in a visual format. Add a new element and find our twitter component to add. There are mainly two layers to the element. The first layer is the parent or the container of feeds, inside which contains the tweets displayed in either list, slider or ticker layout.

List Feeds

Some of the configurable options found in Twitter List Feeds are as follows:
General Setting
  • Fetch Type: Choose between retrieving the tweets from either the username or hashtag value
  • Username: The twitter username from which you wish to fetch it's tweets. You can fetch tweets from multiple users in the same place, just separate the usernames by comma. Leaving this field retrieves tweets from the user whose Twitter API is in use.
  • Hashtag: Fetch tweets on the basis of Hashtag values provided.
  • Total Tweets: The number of tweets to fetch
  • Date and Time format: Whether to show the full date and time, date only or elapsed time of individual tweets publish.
  • Fetch Unavailable Message: Message to show when tweets from particular twitter user or any error occurs with the Twitter API.
Component Setting
  • Show Follow Button: Check it to show follow button of twitter users.
  • Disable Twitter Actions: Check it to hide the Twitter actions(such as Reply, Retweet and Favourite)
  • Disable Username: Check it to hide the username of the twitter user from the tweets.
  • Disable Image of Feeds: Check it to hide the media images from the twitter feeds.
  • Image width: Manipulate the media image size to be displayed in the tweets.
Layout Setting
  • Template: The different predefined templates for displaying tweets in attractive ways.
  • Customization Settings: Color customization of different components of tweets.
